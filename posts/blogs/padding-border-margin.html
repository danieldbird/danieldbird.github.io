Padding, Border, and Margin

7th February 2022

In web development, everything displayed on a page is surrounded by an invisible box.

This is referred to as the box model.

Inside of these simple boxes is all of the content that makes up the web.

In the following examples we will pretend we are wrapping our content in either a generic <div> or generic <span> tag.
Unlike other tags, these do not come with any padding, border or margin applied by default.

Let‚Äôs start from the inside out.

Content

In the centre is the content to be displayed. This can be anything. Text, an image, a video, and so on. Without any styling applied, the invisible box is wrapped tightly around your content. With no spacing between what is inside the box, no spacing between what is outside the box, and no visible border.

Padding

Outside of the content, you have the padding. Padding is what separates the content from the box it is in, and therefore the border. You can not have a negative amount of padding. Any background applied to the box will show up in the padding area. The padding can be set individually, or using shorthand clockwise notation.

Border

On the outside of the padding is the border. It sits between the padding and the margin. The border property values that can be set on a border are size, style and colour. Each side of a box can have different border property values.

Margin

Lastly, on the outside of the border is the margin. This can be thought of as invisible space outside of the box, and it pushes the box away from elements on the outside. Because any background is only applied to the box and it‚Äôs padding, it will not show up in the margin. A margin can have a positive or negative value, which means it is able to overlap other elements if required. The margin can be set individually, or using shorthand clockwise notation.

Box Sizing

If you are using the default box-model box-sizing: content-box;, both the padding and border will be counted towards the width and height of the box. If you prefer a simpler method, you can use box-sizing: border-box;, so you can set your width and height as you would like, and the calculation to minus the padding and border is done for you automatically.
